# 这是什么?
这是一类实现跨版本连接fabric服务器的mod。mc分为客户端和服务端，当玩家连接服务器时，客户端和服务端发生分离，此时原版mc强制客户端和服务端的版本完全相同，例如1.21.7fabric的客户端只能连接1.21.7fabric的服务端。这在游玩中是非常不方便的，因为兼容性的问题，服务端的版本不可能恒定在1.21.7fabric这一个版本。若使用原版连接，则需要根据不同的服务端版本配置不同版本的客户端整合包，这需要进行mod的重新整理和快捷键的迁移，是非常繁琐的。因此，可以使用viafabric这个mod，使用一个固定的客户端整合包，连接多个不同版本的fabric服务器。
